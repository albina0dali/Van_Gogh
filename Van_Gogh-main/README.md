# 🌾 АгроСубсидия KZ

**AI-система скоринга для государственных субсидий Казахстана**

Разработано для **Decentrathon 5.0** | Трек: **AI for Government**

---

## 📋 Описание проекта

**АгроСубсидия KZ** — интеллектуальная система AI-скоринга заявок на государственные субсидии для сельхозпроизводителей Казахстана. Система анализирует параметры хозяйства и выдаёт прозрачную оценку шансов на получение субсидии.

### Решаемая проблема

Сегодня государственные субсидии распределяются по принципу «первый подал — первый получил». Это приводит к:
- Неэффективному использованию бюджетных средств
- Отсутствию учёта реальной эффективности хозяйств
- Непрозрачности процесса для фермеров

**АгроСубсидия KZ** переводит процесс на data-driven подход с объективными критериями оценки.

---

## 🚀 Функционал

- **📊 Дашборд** — обзор системы, ключевые метрики, шкала скоринга
- **📝 Новая заявка** — ввод параметров фермы, AI-скоринг 0–100 баллов, анализ рисков
- **📋 История** — все заявки текущей сессии с фильтрацией и графиками
- **🔬 Аналитика** — распределение баллов, статистика решений
- **💬 AI Консультант** — чат на базе Llama 3.3 70B (Groq), отвечает на казахском и русском

### Скоринг-система

| Баллы | Решение | Описание |
|-------|---------|----------|
| 70–100 | ✅ Ұсынылады | Субсидия рекомендована |
| 40–69 | ⚡ Тексеріс | Требует ручной проверки |
| 0–39 | 🚨 Жоғары тәуекел | Высокий риск, отказ вероятен |

## 📦 Установка и запуск

### Требования
- Python 3.8+
- Groq API ключ (бесплатно: https://console.groq.com)

### 1. Установите зависимости
```bash
pip install -r requirements.txt
```

### 2. Создайте .env файл
```bash
cp .env.example .env
```
Откройте `.env` и добавьте ваш Groq API ключ:
```
GROQ_API_KEY=gsk_ваш_ключ_здесь
```

### 3. Запустите приложение
```bash
streamlit run app.py
```

Откройте браузер: **http://localhost:8501**
