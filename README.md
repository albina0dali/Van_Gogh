# Van_Gogh

ML-система для скоринга заявок на субсидии: от загрузки данных и обучения модели до API и демо-интерфейса.

## Что Умеет Система

- Скоринг заявок и формирование short-list
- Подготовка признаков и обучение модели
- Переобучение и расчет метрик качества
- Объяснение предсказаний
- Визуализация и просмотр результатов в веб-интерфейсе

## Быстрый Старт

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main_pipeline.py
```

После выполнения пайплайна результаты будут сохранены в папке `results/`.

## Как Запустить API

```bash
python api_server.py
```

По умолчанию API поднимается локально и использует файлы модели/трансформеров из проекта.

## Ключевые Файлы

- `main_pipeline.py` — основной pipeline обработки и скоринга
- `api_server.py` — HTTP API для интеграции
- `scoring_engine.py` — логика расчета скорингов
- `model_trainer.py` — обучение и обновление модели
- `demo.py` — локальный сценарий демонстрации
- `templates/index.html` — веб-страница интерфейса
