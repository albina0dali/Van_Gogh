## ⚡ Быстрый запуск (3 шага)

### 1. Распаковать и установить
```bash
unzip SubsiSmartKZ.zip
cd SubsiSmartKZ
./setup.sh
```

### 2. Запустить консольный анализ
```bash
python scoring_engine.py
```

### 3. Или запустить веб-интерфейс
```bash
python app.py
# Откройте в браузере: http://localhost:5000
```

## 🎯 Что делает система

1. **Предобработка** - извлекает признаки из даты, кодирует категории
2. **K-Means** - сегментирует заявителей на 4 группы
3. **Детектирование аномалий** - выявляет подозрительные заявки (3-sigma)
4. **Risk Scoring** - присваивает каждой заявке оценку риска 0-100
5. **Региональный анализ** - показывает эффективность по регионам
6. **Веб-дашборд** - интерактивная визуализация результатов
